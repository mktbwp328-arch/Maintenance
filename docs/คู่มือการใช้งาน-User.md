# 📘 คู่มือการใช้งานระบบแจ้งซ่อม MTD (สำหรับผู้ใช้งาน)

ระบบแจ้งซ่อมออนไลน์ของฝ่ายซ่อมบำรุง (MTD) — แจ้งซ่อมได้ **2 ช่องทาง**:
1. 🖥️ ผ่านเว็บไซต์
2. 📱 ผ่าน LINE

เมื่อแจ้งซ่อมแล้ว ระบบจะแจ้งเตือนกลับให้คุณทราบสถานะ **อัตโนมัติจนงานเสร็จ**

---

## 🖥️ ช่องทางที่ 1: แจ้งซ่อมผ่านเว็บไซต์

### ขั้นที่ 1 — เข้าสู่ระบบ
1. เปิดเว็บ: **https://maintenance-gamma-tan.vercel.app**
2. กรอก **ชื่อผู้ใช้** และ **รหัสผ่าน** ที่ได้รับจากแอดมิน
3. กด **เข้าสู่ระบบ**

### ขั้นที่ 2 — เปิดฟอร์มแจ้งซ่อม
- กดปุ่ม **➕ แจ้งซ่อม** (มุมขวาบน)

### ขั้นที่ 3 — กรอกข้อมูล
| ช่อง | คำอธิบาย | จำเป็น |
|---|---|:---:|
| **ชื่อผู้แจ้ง** | ชื่อของคุณ | ✅ |
| **เบอร์โทร** | เบอร์ติดต่อกลับ | – |
| **อีเมล** | สำหรับรับแจ้งเตือนทางเมล | – |
| **ความเร่งด่วน** | ปกติ / เร่งด่วน / วิกฤต (หยุดการผลิต) | ✅ |
| **เครื่องจักร / อาคาร** | เลือกสิ่งที่ต้องการแจ้งซ่อม | ✅ |
| **หมวดปัญหา** | เช่น ไฟฟ้า / เครื่องกล / ระบบลม | – |
| **อาการ / รายละเอียดปัญหา** | อธิบายอาการที่พบ | ✅ |
| **รูปภาพ** | แนบรูปจุดที่เสีย (กดเลือกไฟล์) | – |

### ขั้นที่ 4 — ส่ง
- กด **ส่งใบแจ้งซ่อม**
- ระบบจะแสดงเลขที่ใบ เช่น `MTD-26060022` และแจ้งฝ่ายซ่อมบำรุงอัตโนมัติ

### ขั้นที่ 5 — ติดตามสถานะ
- หน้าแรกจะแสดง **รายการที่คุณแจ้ง** พร้อมสถานะปัจจุบัน
- กดที่ใบเพื่อดูรายละเอียด/ประวัติการดำเนินการ

---

## 📱 ช่องทางที่ 2: แจ้งซ่อมผ่าน LINE

### ⚠️ ทำครั้งแรกครั้งเดียว — เพิ่มเพื่อนบอท
ก่อนใช้งาน ต้อง **เพิ่มเพื่อน** บัญชีทางการ (OA) ของระบบก่อน เพื่อให้บอทแจ้งเตือนกลับหาคุณได้:
1. เปิดแอป **LINE** → แท็บ **หน้าหลัก** → ช่องค้นหา
2. พิมพ์ **`@743cmohm`**
3. กด **เพิ่มเพื่อน** บัญชี **BWP แจ้งซ่อม**

> 📌 ถ้าไม่เพิ่มเพื่อน บอทจะ**ส่งแจ้งเตือนกลับหาคุณไม่ได้** (เป็นกฎของ LINE)

### ขั้นที่ 1 — เปิดฟอร์มแจ้งซ่อม
- เปิดลิงก์: **https://liff.line.me/2010375570-jvQMAOCL**
- (หรือกดลิงก์/เมนูที่ฝ่ายซ่อมบำรุงแชร์ไว้ในกลุ่ม)

### ขั้นที่ 2 — กรอกข้อมูล
ฟอร์มจะดึง **ชื่อ LINE ของคุณ** มาให้อัตโนมัติ จากนั้นเลือก/กรอก:
- **เครื่องจักร / อาคาร** ที่ต้องการแจ้งซ่อม
- **หมวดปัญหา** (ถ้ามี)
- **ความเร่งด่วน**
- **อาการ / รายละเอียดปัญหา**
- **รูปภาพ** — กดเลือกรูป หรือ **ถ่ายรูปสด** จากกล้องได้เลย

### ขั้นที่ 3 — ส่ง
- กด **ส่งแจ้งซ่อม**
- บอทจะส่ง **การ์ดยืนยัน** กลับมาในแชต: ✅ *รับแจ้งซ่อมของคุณแล้ว*

---

## 🔔 การแจ้งเตือนสถานะ (ทั้ง 2 ช่องทาง)

หลังแจ้งซ่อม คุณจะได้รับ **การ์ดแจ้งเตือนทาง LINE** ทุกครั้งที่สถานะเปลี่ยน โดยสีแถบบนการ์ดบอกสถานะ:

| สี | สถานะ | ความหมาย |
|---|---|---|
| 🟡 เหลือง | แจ้งซ่อม | รับเรื่องแล้ว รอดำเนินการ |
| 🔵 ฟ้า | ดำเนินการ | ช่างกำลังซ่อม |
| 🟣 ม่วง | ส่งซ่อม / รออะไหล่ | รออะไหล่หรือส่งซ่อมภายนอก |
| 🟢 เขียว | สำเร็จ | ซ่อมเสร็จแล้ว |
| 🔴 แดง | ยกเลิก | ยกเลิกใบแจ้งซ่อม |

### เมื่อซ่อมสำเร็จ การ์ดจะแสดงเพิ่ม:
- 🛠️ **ผู้ซ่อม** (ชื่อช่างที่ดำเนินการ)
- 🔧 **รายละเอียดการซ่อม** (สาเหตุ/วิธีแก้ไข)
- 📅 **วันที่สำเร็จ**
